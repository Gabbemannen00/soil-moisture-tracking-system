
**Description

This map includes the setup of the Flask server and SQLite database on the Raspberry Pi Zero 2 W

---

**Content 

*app.py* = The Flask server that is responsible to receive current moisture data from the Raspberry Pi Pico W

*moisture_db* = The SQLite database that is connected to the Flask server, receives data from the server and stores it in the table of the database.

---

**Future work

*Add visualization* = Implement a simple dashboard application in Flask to visualize the moisture data in real time.
